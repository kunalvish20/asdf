# Portfolio\n\nThis repository contains my personal portfolio built with React, Vite, and Tailwind CSS. The production build lives under dist/.\n\n## Getting Started\n\n1. Install deps: 
pm install\n2. Start dev server: 
pm run dev\n3. Build for production: 
pm run build\n\n## Deployment\n\nDeploy the contents of dist/ to your hosting provider or use GitHub Pages / Vercel.
